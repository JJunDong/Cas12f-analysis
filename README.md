# Cas12f-analysis
screen CRISPR-Cas assay


1. First, install R. 

2. Then run “perl /path/your_folder_path/search_rep.pl xxx.fa xxx.final.txt”

Ensure the script file and sequence file are located in the same path to facilitate execution.

Where path/your_folder_path should be replaced with the path to your own folder; xxx.fa is the DNA sequence file containing the coding reading frames (CRFs) of candidate protein IDs along with 5 kb of upstream and downstream flanking sequences; and xxx.final.txt is the name of the corresponding output file.
